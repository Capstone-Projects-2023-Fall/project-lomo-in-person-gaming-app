Backend Unit Tests

Laravel has a built-in test suite wrapper that uses the PHPUnit testing framework to create and run all tests.

The unit tests are grouped together under Feature tests. Each feature test corresponds to a use case.

note

The Laravel Sail docker container must be running for the commands to work. For more testing commands, check out the Larvel 10 testing documentation. Remember to use the sail or docker command to execute any Laravel commands.

To run ALL unit tests, run the following command:

sail artisan test

To run ALL unit tests AND get test coverage results, run the following command:

sail artisan test --coverage

3. Beacon Creation Test​

Test: Checks that POST request to /api/beacons/ successfully accepts JSON data​

test_post_beacon_request_returns_successful_response()

  • Returns: HTTP status code 201 for successful resource creation

Test: Checks that POST request to /api/beacons/ fails when required fields are empty​

test_post_beacon_request_fails_when_required_fields_are_empty()
